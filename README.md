# 📊 Training Dashboard - การประปาภูมิภาค

## 🎯 รายละเอียดโครงการ
Dashboard สำหรับการวางแผนและติดตามข้อมูลการฝึกอบรมของการประปาภูมิภาค ปี 2569

## ✨ คุณสมบัติ
- 📈 แสดงสถิติการฝึกอบรมแบบ Real-time
- 🔍 ระบบกรองและค้นหาข้อมูล
- 📱 Responsive Design รองรับทุกอุปกรณ์
- 🎨 UI/UX ที่สวยงามและใช้งานง่าย
- 📊 จัดกลุ่มตามโครงการหลัก/โครงการย่อย
- 🔄 Modal แสดงรายละเอียดแบบเต็มหน้าจอ

## 🚀 การใช้งาน
1. เปิดไฟล์ `training-dashboard.html` ในเบราว์เซอร์
2. ใช้ตัวกรองด้านบนเพื่อค้นหาข้อมูล
3. คลิกที่การ์ดโครงการเพื่อดูรายละเอียด
